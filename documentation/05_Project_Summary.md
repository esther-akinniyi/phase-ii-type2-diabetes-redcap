# Project Summary

## Overview

This project demonstrates the complete design, implementation, validation, and documentation of a REDCap Electronic Data Capture (EDC) database for a simulated Phase II Type 2 Diabetes clinical trial.

The project was developed to demonstrate practical Clinical Data Management (CDM) skills using REDCap.

---

# Project Deliverables

The following components were successfully completed:

- REDCap database configuration
- Six electronic Case Report Forms (eCRFs)
- Required field validation
- Numeric validation
- Date validation
- Calculated BMI field
- User Acceptance Testing (UAT)
- Data Quality Review
- Audit Trail Review
- Participant Demographics Report
- CSV Data Export
- Data Dictionary Export
- Project Documentation
- GitHub Portfolio Repository
